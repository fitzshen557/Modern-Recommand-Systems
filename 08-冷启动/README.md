# 冷启动（Cold Start）

> 冷启动是推荐系统的"死亡谷"：新用户/新物品缺乏交互数据，协同过滤无从下手。
> 核心策略三条线：**内容特征 / 迁移学习 / 元学习**

---

## 冷启动问题分类

```
冷启动
├── 新用户冷启动（User Cold Start）
│   ├── 无历史行为 → 靠注册信息/人口属性
│   └── 少量行为 → 快速个性化（meta-learning）
└── 新物品冷启动（Item Cold Start）
    ├── 上线即需曝光 → 靠内容特征
    └── 积累少量交互 → 快速收敛（热身策略）
```

## 2024-2025 新趋势

**LLM 成为冷启动新主力**：
- LLM 拥有海量世界知识，对新物品（哪怕没有任何交互）也能从文本描述推断其属性和潜在受众
- 典型路径：`新物品文本 → LLM 生成特征/标签 → 传统推荐模型使用`
- 代表工作：KAR（2023）、GoRec（2024）、ColdLLM（2024）
