## 📄 arXiv 新论文（2026年7月，cs.IR）

### 1. Autonomous Information Seeking: A Roadmap for Agentic Recommender Systems
- **链接**: https://arxiv.org/abs/2607.04433
- **摘要**: 提出 Agentic 推荐系统的统一分类框架，涵盖三大范式：agent-assisted recommendation、agent-as-recommender、agent-as-user-simulator。沿自主性程度组织方法，分析 agent 如何增强 profile、memory、tool use、workflow 等关键组件。
- **关注点**: 🌟 综述论文，LLM Agent + 推荐系统的系统性路线图，对理解行业趋势非常有价值

### 2. Towards Mitigating Length Bias in Large Language Models for Recommendation (LBR)
- **链接**: https://arxiv.org/abs/2607.04270
- **摘要**: 发现 LLM 推荐系统中的长度偏差问题——输入端长文本 item 占据更多 token 获得过多注意力，输出端自回归解码不利于长 item。提出 LBR 框架，轻量且模型无关。
- **关注点**: 🌟 实际工程中常见问题，LLM-based recommender 的核心偏差分析和解决方案

### 3. Preference-Optimized Retrievers for Tool Selection with LLMs (PORTS)
- **链接**: https://arxiv.org/abs/2607.05441
- **摘要**: 提出偏好优化方法训练检索器用于 LLM 工具选择，通过困惑度启发的偏好信号微调检索器，优化选择概率与下游性能的相关性。
- **关注点**: 检索+LLM 工具调用的对齐方法，对推荐系统中工具/RAG 模块选择有参考价值

### 4. Modality Relevance is not Modality Utility: Post-hoc Selective Modality Escalation for Cost-Aware Multimodal RAG
- **链接**: https://arxiv.org/abs/2607.05438
- **摘要**: 揭示多模态RAG中"模态相关性≠模态实用性"——预检索路由器会过度升级视觉模型。提出后验选择性模态升级策略：先用文本/表格低成本回答，验证失败再升级视觉模型。
- **关注点**: 多模态推荐/检索场景的成本优化思路

### 5. Object-Evidence Preserving Token Merging for Vision-Language Retrieval (SaMer)
- **链接**: https://arxiv.org/abs/2607.04605
- **摘要**: 提出对象感知 token 合并框架，将图像侧 token 压缩到 K 个代表质心，去除 93%+ token 的同时保持 late-interaction 接口，存储减少16倍。
- **关注点**: 多模态检索的效率优化，可用于图文推荐场景
